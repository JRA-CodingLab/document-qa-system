# Beispiel-Markdown-Dokument

Dies ist ein Beispiel-Markdown-Dokument zum Testen.

## Überblick

Der RAG-Dokumentenassistent verarbeitet Dokumente in verschiedenen Formaten
und beantwortet Fragen basierend auf deren Inhalt. Diese Markdown-Datei
zeigt die Fähigkeit des Systems, `.md`-Dateien zu verarbeiten.

## Schlüsselkonzepte

### Retrieval-Augmented Generation

RAG kombiniert Dokumentenabruf mit Sprachmodell-Generierung. Anstatt sich
ausschließlich auf die Trainingsdaten des Modells zu verlassen, rufen
RAG-Systeme relevanten Kontext aus einer Dokumentensammlung ab, bevor
sie eine Antwort generieren.

### Vektoreinbettungen

Text wird in numerische Vektoren umgewandelt, die semantische Bedeutung
erfassen. Ähnliche Texte erzeugen Vektoren, die im Einbettungsraum
nah beieinander liegen, was eine effiziente Ähnlichkeitssuche ermöglicht.
