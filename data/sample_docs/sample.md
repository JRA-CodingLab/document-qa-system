# Sample Markdown Document

This is a sample markdown document used for testing.

## Overview

The Document QA System processes documents in multiple formats and
answers questions based on their content. This markdown file demonstrates
the system's ability to handle `.md` files.

## Key Concepts

### Retrieval-Augmented Generation

RAG combines document retrieval with language model generation. Instead of
relying solely on the model's training data, RAG systems retrieve relevant
context from a document collection before generating an answer.

### Vector Embeddings

Text is converted into numerical vectors that capture semantic meaning.
Similar texts produce vectors that are close together in embedding space,
enabling efficient similarity search.
